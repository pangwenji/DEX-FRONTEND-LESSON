# MetaNodeSwap DEX 前端业务教学（5 课时）


## 课程概述

本课程通过一个可运行的 Next.js DEX 前端，串联代币部署、流动性池、流动性管理、代币交换和池子详情。每课包含学习目标、概念讲解、代码导读、课堂实操、检查点与课后作业。
> **⚠️ 消化吸收的前提是同学需要完成前面 MetaNodeSwap 项目的合约课程学习**

## 使用方式

1. 按课时进入对应目录阅读 `README.md`；
2. 在该课的 `code` 目录安装依赖并启动项目；
3. 按“代码导读 → 实操任务 → 课堂检查点”完成学习；
4. 所有链上操作默认使用 Sepolia，请准备测试 ETH，切勿使用真实资产或提交私钥。
